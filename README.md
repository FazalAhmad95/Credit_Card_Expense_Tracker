Description:
This project focuses on analyzing monthly credit card spending patterns to track expenses, identify major spending categories, and improve financial planning. The dashboard provides a clear overview of where and how money is spent over time.

Key Highlights:

Cleaned and organized transaction data using Excel and Power Query for accurate analysis.

Built an interactive dashboard showing total monthly expenses, category-wise spending, and payment mode distribution.

Analyzed month-over-month expense trends to identify high-spending periods and cost patterns.

Enabled better budgeting decisions by highlighting unnecessary or high-cost expense categories.

Tools & Technologies:
Power BI, MS Excel, Power Query
